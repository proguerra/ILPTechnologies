# ILP Technologies — Simple Informational Site

This repo contains a single-page static website highlighting ILP Technologies and its SaaS portfolio.

## Structure
```
ilp-technologies-site/
├── index.html
└── assets/
    └── img/
        ├── clearcomply-hr-logo.png
        └── ordersign-md.png
```

## Local preview
Open `index.html` in your browser.

## Deploy to GitHub Pages
1. Create a new repo named `ilp-technologies-site` (or any name).
2. Upload all files in this folder to the root of the repo.
3. In GitHub, go to **Settings → Pages**.
4. Under **Source**, choose **Deploy from a branch**, then select `main` and `/ (root)` and save.
5. Your site will be available at `https://<your-username>.github.io/<repo-name>/` after it builds.

## Update contact email
Open `index.html` and search for `mailto:` to change the destination address.
